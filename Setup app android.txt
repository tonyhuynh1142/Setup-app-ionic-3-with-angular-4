1. Install Java JDK
http://www.oracle.com/technetwork/java/javase/downloads/jdk9-downloads-3848520.html
- Downloading
- Click to install it
2. Install Android SDK
https://developer.android.com/studio/index.html
- Downloading: Select a different platform (Android Studio package)
- Click to install it
3. Environment Variale
- Get JAVA_HOME path
- Setup JAVA_HOME to Environment
- Get ANDROID_HOME path
- Setup ANDROID_HOME to Environment
	+ User variables for administrator:
		- path: C:\Users\Administrator\AppData\Local\Android\Sdk\platform-tools;C:\Users\Administrator\AppData\Local\Android\Sdk\tools
	+ Systemvariables:
		- Path: ;%Sdk%\tools;%ANDROID_HOME%\platform-tools;%ANDROID_HOME%\build-tools

4.Install NodeJS
- Click to install it
- Downloading
- Click to install it
5. Install Cordova version 6.3.1
- Windows key + R => cmd
- npm install -g cordova@6.3.1 ionic
- test version cordova

- Windows key + R => cmd
